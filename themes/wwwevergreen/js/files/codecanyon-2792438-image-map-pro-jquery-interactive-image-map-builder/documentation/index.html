Working with the Editor

Top

Take the tour!

When you open the plugin's admin page for the first time, you will be asked to take a guided tour to get familiar with the Editor. We highly recommend doing that and you will get comfortable with the Editor much quicker.

Creating a new image map

To create a new image map, click the New button in the top-left corner of the Editor, enter a name and click Create.
Then you need to click the Choose Image... button and use the standard WordPress Media Library to insert an image by uploading, entering an URL, or choosing an existing image.

Responsive

The image map is responsive! If you check the Responsive checkbox, the image map will scale to 100% width of its parent element, and the Width and Height settings are used only to determine the aspect ratio of the image map.

Shortcode

On the left side of the Editor, there is an input field where you can enter a shortcode for the image map. If you enter my-shortcode, then you need to insert [the-shortcode] in your page or blog post. That's all!

Drawing Shapes

To start drawing shapes, first make sure that you are in Edit mode, by clicking the button on the top-right of the Editor.
Then, simply select a tool from the toolbar on the left and either click, or click and drag the image, depending on the tool that you selected.
Above the image you should see helpful tips that will give you an idea how the controls work.

Styles

To change the style of a shape, like color or opacity, first make sure that the shape is selected. You can select a shape by clicking it in the shapes list (on the right), or clicking it in Edit Mode.
Then, you can scroll down and change various settings under Default Style and Mouseover Style. Experiment with different settings and use what you like!

Preview Mode

At any point you can click the "Preview Mode" button in the top-right side of the Editor to see how the image will look like in the end. You can still make changes to the settings and the preview will instantly update itself.

Save, Load

Don't forget to save! You can create as many image maps as you wish, and you can switch between them by clicking "Load" and choosing the image map that you saved previously.

Editing an instance

To start editing an instance, click the Edit button next to it's name. The settings for this instance will load and you can edit them. Don't forget to press Save Changes when you are done editing!
